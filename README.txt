DriveX — Single-page Car Rental (School project)
-----------------------------------------------
Files:
- index.html    : Main HTML file
- style.css     : Styling
- script.js     : Data and interactivity

Project goals:
- Build a simple, responsive single-page web app using HTML, CSS, JavaScript.
- Provide a dataset of cars (6 items).
- Add interactive features: search & brand filter.
- Keep code clean and easy to explain.

How to run:
1. Open index.html in a browser.
2. Use the search box and filter buttons to test.
3. Click "View" for details or "Book Now" to run demo booking flow (prompt + alert).

Versioning (suggested commits for Git):
1) initial commit: add index.html
2) add style.css and basic layout
3) add script.js and dataset
4) add filters & search functionality
5) final polish & README

Teacher notes:
- This is a frontend-only demo (no backend).
- All JavaScript is client-side. Data is a JS array in script.js.
